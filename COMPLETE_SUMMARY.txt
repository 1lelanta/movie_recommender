📚 COMPLETE API REFERENCE
═════════════════════════

MovieRecommender Class Methods:

Public Methods:
───────────────

1. __init__(movies_path: str, ratings_path: str) → None
   Purpose: Initialize and build the recommendation engine
   Auto-calls: _load_data(), _preprocess_data(), _compute_similarity_matrix()

2. recommend_movies(movie_title: str, top_n: int = 10) → List[Tuple[str, float, str]]
   Purpose: Get top N recommendations based on genre similarity
   Returns: [(title, score, genres), ...]
   Raises: ValueError if movie not found
   Example:
      recs = recommender.recommend_movies('Inception', top_n=5)
      # [(Interstellar, 0.82, 'Adventure|Drama|Sci-Fi'), ...]

3. get_movie_info(movie_title: str) → Optional[Dict]
   Purpose: Get movie metadata
   Returns: {'title': '...', 'genres': '...', 'movieId': int} or None
   Example:
      info = recommender.get_movie_info('Avatar (2009)')
      # {'title': 'Avatar (2009)', 'genres': 'Action|Adventure|Sci-Fi', 'movieId': 19995}

4. search_movies(query: str) → List[str]
   Purpose: Find movies by partial title match (case-insensitive)
   Returns: [title1, title2, ...] matching query
   Example:
      matches = recommender.search_movies('Matrix')
      # ['The Matrix (1999)', 'The Matrix Reloaded (2003)', ...]

5. get_all_movies() → List[str]
   Purpose: Get list of all movies in dataset
   Returns: [title1, title2, ...] for all movies
   Use for: Understanding dataset size, validation

6. display_recommendations(recommendations: List[Tuple], reference_movie: str = "") → None
   Purpose: Pretty-print recommendations in table format
   Parameters:
      - recommendations: List of (title, score, genres) tuples
      - reference_movie: Optional movie name for header
   Output:
      ================================================================================
      Recommendations based on: The Matrix
      ================================================================================
      
      Rank   Movie Title                                        Similarity   Genres
      --------------------------------------------------------------------------------
      1      The Matrix Reloaded (2003)                        0.8234       Action|Sci-Fi
      ...

💡 ALGORITHM EXPLANATION
═════════════════════════

How the system works in 5 steps:

Step 1: DATA LOADING
──────────────────
Input:  movies.csv (movieId, title, genres)
        ratings.csv (userId, movieId, rating)

Output: movies_df (9000+ movies)
        ratings_df (100,000+ ratings)

Example Data:
  movieId | title               | genres
  --------|---------------------|----------------------------
  1       | Toy Story (1995)    | Adventure|Animation|Children
  2       | Jumanji (1995)      | Adventure|Children|Fantasy

Step 2: DATA PREPROCESSING
──────────────────────────
Input:  Raw DataFrames with potential issues

Tasks:
  • Fill missing genres → "Unknown"
  • Remove movies with no title
  • Remove duplicate entries
  • Reset indices for consistency

Output: Clean, validated DataFrames ready for analysis

Step 3: TF-IDF VECTORIZATION
────────────────────────────
What:   Convert genre strings to numerical vectors
Why:    Allows mathematical comparison

Example:
  "Action|Sci-Fi|Thriller" →  [0.45, 0.32, 0.38, 0.22, ...]
  "Action|Adventure"       →  [0.51, 0.28, 0.12, 0.18, ...]
  "Comedy|Romance"         →  [0.05, 0.08, 0.42, 0.35, ...]

Method: TfidfVectorizer with character n-grams (1-2)
Output: Sparse matrix (9000 × 100 features)

Step 4: COSINE SIMILARITY
─────────────────────────
What:   Calculate similarity between all movie pairs
How:    similarity(A, B) = (A · B) / (||A|| × ||B||)
Range:  0 (completely different) to 1 (identical)

Result: 9000 × 9000 similarity matrix
        similarity_matrix[i][j] = how similar movie i and j are

Example:
  Matrix[0][1] = 0.82  # Movie 1 and 2 are 82% similar
  Matrix[0][5] = 0.25  # Movie 1 and 6 are 25% similar

Step 5: RECOMMENDATION GENERATION
──────────────────────────────────
Input:  movie_title = "Inception"

Process:
  1. Find index of "Inception" → index = 2345
  2. Get similarity scores → [0.82, 0.91, 0.75, 0.88, ...]
  3. Sort indices by score (descending)
  4. Exclude the original movie
  5. Return top 10

Output: [(Interstellar, 0.91, 'Adventure|Sci-Fi'),
         (The Prestige, 0.88, 'Drama|Mystery'),
         ...]
