INSTALLATION STEPS
==================

1. Navigate to project directory:
   cd movie_recommender

2. Install dependencies:
   pip install -r requirements.txt

3. Download MovieLens dataset:
   - Visit: https://grouplens.org/datasets/movielens/
   - Download "MovieLens Latest Small" (easiest option)
   - Extract ZIP file
   - Copy movies.csv and ratings.csv to data/raw/

4. Run the application:
   python main.py

5. Enjoy recommendations!
